# Descrição

Aplicação criada através do GoStack Bootcamp da Rocketseat, o objetivo é buscar, listar e visualizar repositórios do Github atráves do autor/nome.

<img src="https://im3.ezgif.com/tmp/ezgif-3-c5bde84d5735.gif" title="github explorer" />

# Este projeto utiliza:

- Typescript
- React
- React Hooks
- styled-components
- Axios

# Como iniciar:

- `yarn install` para instalar todos os pacotes.
- `yarn start` para iniciar o projeto local.
